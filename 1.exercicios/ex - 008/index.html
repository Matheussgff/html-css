Texto marcado

Podemos criar um texto marcado usando a tag MARK.

Ao utilizar as css para mudar a cor de fundo do marca texto(MARK) ela é adicionada entre os colchetes angulares dessa forma: <mark style="background-color: ex.:lime;"> txt marcado <mark>

E no outro parágrafo, temos outro texto marcado no final.

Para mudar as cores dos marca textos todos do site usa-se as css no head do documento da seguinte forma: style + Mark + background-color.
