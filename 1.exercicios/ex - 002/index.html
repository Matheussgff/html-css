Parágrafos e quebras de linha

Tudo que escrever tem que estar no meio da tag <p> e </p>.

Para quebrar o texto use a tag br
nesse momento há uma quebra de linha no site.

A tag hr

cria uma linha horizontal no código.

Para colocar o sinal de maior e menor no site utiliza as tags &+lt para menor que (less than) <, &+gt para maior que (greater than) >
